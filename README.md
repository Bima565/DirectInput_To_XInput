# Fix_L1

## Overview
This project addresses and fixes the rapid button problem in the L1 DirectInput controller button. It provides a solution for converting DirectInput signals to XInput format to ensure proper button response handling.

## Project Details
- **Language**: C#
- **Type**: Controller Input Fix
- **Purpose**: Fix rapid/repeated button press issues with L1 DirectInput button

## Repository Structure
- `Fix_L1.slnx` - Visual Studio solution file
- `Fix_L1/` - Main project directory containing the implementation
